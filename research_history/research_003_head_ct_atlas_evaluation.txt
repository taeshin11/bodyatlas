OPEN-SOURCE HEAD CT ATLAS DATASET SURVEY
================================================================================

1. CADS (Comprehensive Anatomy Dataset for Segmentation)
   URL:       https://github.com/murong-xu/CADS
   License:   CC BY 4.0
   Format:    NIfTI
   Structures: 167 anatomical structures (full body, head-neck included)
   Scale:     22,022 CT scans from 100+ imaging centers, 16 countries
   Head coverage: comprehensive (skull, brain regions, sinuses, vessels, glands)
   Verdict:   BEST OPTION — most structures, best license, largest scale
   To use:    Download reference CT + seg from HuggingFace, run pipeline

2. TotalSegmentator v2
   URL:       https://github.com/wasserth/TotalSegmentator
   License:   Apache 2.0 (tool), CC BY 4.0 (dataset)
   Format:    NIfTI
   Structures: 117 total, with dedicated head tasks:
               - brain_structures (subcortical regions)
               - head_glands_cavities (sinuses, orbits, glands)
               - headneck_bones_vessels (detailed skull, vessels)
               - craniofacial_structures (facial bones)
   Scale:     1,228 CT images (training/val/test splits)
   Note:      CURRENT DATA IS FROM TotalSegmentator v1 (96 structs)
   Verdict:   UPGRADE PATH — run TotalSegmentator v2 with all head tasks
              on same or better reference CT → get 117+ head structures

3. HaN-Seg (Head and Neck Segmentation)
   URL:       https://zenodo.org/records/7442914
   License:   CC BY-NC-ND 4.0 (NON-COMMERCIAL, no derivatives)
   Format:    NIfTI binary masks
   Structures: 30 organs-at-risk (radiotherapy planning specific)
               brain, brainstem, eyes, optic nerves, chiasm, cochlea,
               parotid, submandibular, thyroid, larynx, pharynx, spinal cord
   Scale:     42 patients (CT + MR)
   Verdict:   PASS — license too restrictive for a free atlas service

4. CT-ORG (TCIA)
   URL:       https://www.cancerimagingarchive.net/collection/ct-org/
   License:   TCIA non-commercial
   Structures: 6 only (liver, lungs, bladder, kidney, bones, brain)
   Verdict:   PASS — too few structures

5. SAROS
   URL:       https://github.com/UMEssen/saros-dataset
   License:   CC BY 4.0
   Structures: 13 body regions (sparse, only every 5th slice labeled)
   Verdict:   PASS — too sparse for atlas use

6. SPL Head and Neck Atlas (Open Anatomy Project)
   URL:       https://www.openanatomy.org/atlas-pages/atlas-spl-head-and-neck.html
   License:   3D Slicer license (liberal open source)
   Format:    NRRD/VTK/STL/OBJ (no paired CT images)
   Coverage:  skull, mandible, spine, neck muscles, vessels, glands
   Verdict:   LIMITED — no paired CT images, model-based not image-based

VERDICT — SHOULD HEAD-CT BE REPLACED?
================================================================================

Current data quality: ACCEPTABLE but improvable
  + 96 structures, 262 slices, real labels, no empty slices
  + TotalSegmentator v1 quality (well-established pipeline)
  - Missing ~20+ head-specific structures vs TotalSegmentator v2
  - Window setting (soft tissue only) — can't see skull detail well
  - info.json bug (now fixed)

RECOMMENDATION: Replace with TotalSegmentator v2 output (2 options)

  OPTION A — TotalSegmentator v2 upgrade (HIGH VALUE, needs GPU time ~30min)
    1. Download a clean head CT from TCIA (e.g., HNSCC-3D-CT, CT-HEAD-NECK)
    2. Install: pip install TotalSegmentator (CUDA available → fast)
    3. Run all head tasks: totalsegmentator -i head.nii.gz -o seg/ --task total_mr
       + task head_glands_cavities
       + task headneck_bones_vessels
       + task craniofacial_structures
    4. Merge seg masks → run existing data_pipeline → 117+ structures
    Expected improvement: +21 head structures, better label coverage

  OPTION B — CADS reference CT (HIGHEST VALUE, more work)
    1. Download from HuggingFace: mrmrx/CADS-dataset
    2. Pick representative head-neck CT case
    3. 167 structures including fine-grained brain/head regions
    Blocker: Need to verify individual case CT quality and coverage

  OPTION C — Keep current, only fix the bug (DONE)
    - info.json bug fixed (203→262)
    - Current 96 structures are usable
    - Defer atlas replacement to next sprint

IMMEDIATE ACTION TAKEN: Fixed info.json bug (Option C)
